# 企业前端 UX 评审检查表

## 1. 任务效率

- [ ] 高频任务在 3 次点击内可达
- [ ] 列表页可完成主要操作（无需频繁跳详情）
- [ ] 默认排序/筛选符合日常工作流

## 2. 可理解性

- [ ] 信息分组符合业务心智模型
- [ ] 主次按钮层级明确
- [ ] 状态标签语义一致且可区分

## 3. 防错与恢复

- [ ] 危险操作有明确确认与影响说明
- [ ] 错误提示提供可执行修复建议
- [ ] 可撤销或可回滚路径已定义

## 4. 一致性

- [ ] 与现有组件库和页面模式一致
- [ ] 相同业务对象在不同页面命名一致
- [ ] 颜色、图标、文案语气一致

## 5. 边界与异常

- [ ] 空数据、加载中、超时、失败状态齐全
- [ ] 权限不足时有清晰提示
- [ ] 大数据量场景可用（分页/虚拟滚动/延迟加载）
